Shortcuts

  • Tournament

    CTRL+N
    New Tournament
    CTRL+O
    Open Tournament

  • All lists

    Any character
    Search for entry which starts with the typed characters
    F3
    Continue search with current search string
    ALT+Enter
    Edit selected item
    Insert
    Add new entry (if adding is possible)
    Delete
    Delete selected entries (if deleting possible)
    CTRL+A
    Select all entries (if selecting multiple entries is possible)
    CTRL+I
    Invert selection (if selecting multiple entries is possible)

  • List of players

    ALT+E
    Show the events player has entered

  • Edit Ranking

    ALT+Up Arrow
    Move entry up
    ALT+Down Arrow
    Move entry down

  • Match Results and Match Times

    ALT+Right Arrow
    Go to next round
    ALT+CTRL+Right Arrow
    Go to last round
    ALT+Left Arrow
    Go to previous round
    ALT+CTRL+Left Arrow
    Go to first round

  • Match Overview

    CTRL+T
    Add new tab (Overview)
    CTRL+W
    Remove current tab (Overview)
    ALT+Right Arrow
    Go to next tab (Overview)
    ALT+Left Arrow
    Go to previous tab (Overview)
    ALT+1, ..., 9
    Go to tab (Overview) n
    F5
    Refresh overview
    CTRL+P
    Print overview
    CTRL+SHIFT+P
    Print overview, but without showing a print dialog. Same for SHIFT+Click "Print"

  • Edit Result (Games)

    Any positive number
    Player A has won the game to number
    Any negative number
    Player A has lost the game to number
    0
    Player A has won the game 11 : 0
    -0
    Player A has lost the game 0 : 11

  • Edit Match Time

    # in table field
    Show the next free table if date and time were entered before

  • Print Result Forms

    CTRL+P
    Print selected groups
    CTRL+SHIFT+P
    Print selected groups, but without showing a print dialog. Same for SHIFT+Click "Print"

  • Print Reports

    CTRL+P
    Print selected report

  • SQL Editor

    CTRL+E
    Execute query
    CTRL+P
    Print result of query
    CTRL+S
    Save result of query to file
